Creating an Interface

gradio.Interface(fn, inputs, outputs, live=False, capture_session=False, title=None, description=None, examples=None)

Interfaces are created with Gradio using the gradio.Interface() function.

Parameters

  • fn - the function to wrap an interface around.
  • inputs - a single Gradio input component, or list of Gradio input components. Components can either be passed as instantiated objects, or referred to by their string shortcuts. The number of input components should match the number of parameters in fn.
  • outputs - a single Gradio output component, or list of Gradio output components. Components can either be passed as instantiated objects, or referred to by their string shortcuts. The number of output components should match the number of values returned by fn.
  • live - whether the interface should automatically reload on change.
  • capture_session - if True, captures the default graph and session (needed for Tensorflow 1.x)
  • title - a title for the interface; if provided, appears above the input and output components.
  • description - a description for the interface; if provided, appears above the input and output components.
  • examples - sample inputs for the function; if provided, appears below the UI components and can be used to populate the interface. Should be nested list, in which the outer list consists of samples and each inner list consists of an input corresponding to each input component.

gradio.Interface.launch(share=False)

Generates the UI for the interface.

Parameters

  • share - whether to create a publicly shareable link from your computer for the interface.

Input Components

gradio.inputs.Textbox(self, lines=1, placeholder=None, default=None, numeric=False, label=None)

Component creates a textbox for user to enter input. Provides a string (or number is is_numeric is true) as an argument to the wrapped function.

Parameters

  • lines - number of line rows to provide in textarea.
  • placeholder - placeholder hint to provide behind textarea.
  • default - default text to provide in textarea.
  • numeric - whether the input should be parsed as a number instead of a string.
  • label - component name in interface.

String Shortcuts

  • "text" - default implementation.
  • "textbox" - sets lines = 7.
  • "number" - sets is_numeric = True.

gradio.inputs.Slider(self, minimum=0, maximum=100, default=None, label=None)

Component creates a slider that ranges from minimum to maximum. Provides a number as an argument to the wrapped function.

Parameters

  • minimum - minimum value for slider.
  • maximum - maximum value for slider.
  • default - default value.
  • label - component name in interface.

String Shortcuts

  • "slider" - default implementation.

gradio.inputs.Checkbox(self, label=None)

Component creates a checkbox that can be set to True or False. Provides a boolean as an argument to the wrapped function.

Parameters

  • label - component name in interface.

String Shortcuts

  • "checkbox" - default implementation.

gradio.inputs.CheckboxGroup(self, choices, label=None)

Component creates a set of checkboxes of which a subset can be selected. Provides a list of strings representing the selected choices as an argument to the wrapped function.

Parameters

  • choices - list of strings that represent each checkbox label.
  • label - component name in interface.

String Shortcuts

None

gradio.inputs.Radio(self, choices, label=None)

Component creates a set of radio buttons of which only one can be selected. Provides string representing selected choice as an argument to the wrapped function.

Parameters

  • choices - list of strings that represent each radio label.
  • label - component name in interface.

String Shortcuts

None

gradio.inputs.Dropdown(self, choices, label=None)

Component creates a dropdown of which only one can be selected. Provides string representing selected choice as an argument to the wrapped function.

Parameters

  • choices - list of strings that represent each dropdown element.
  • label - component name in interface.

String Shortcuts

None

gradio.inputs.Image(self, shape=(224, 224), image_mode='RGB', label=None)

Component creates an image upload box with editing capabilities. Provides numpy array of shape (width, height, 3) if image_mode is "RGB" as an argument to the wrapped function. Provides numpy array of shape (width, height) if image_mode is "L" as an argument to the wrapped function.

Parameters

  • shape - tuple representing width and height to crop image to.
  • image_mode - "RGB" if color, or "L" if black and white.
  • label - component name in interface.

String Shortcuts

  • "image" - default implementation.

gradio.inputs.Sketchpad(shape=(28, 28), invert_colors=True, flatten=False, label=None)

Component creates a sketchpad for black and white illustration. Provides numpy array of shape (width, height) as an argument to the wrapped function.

Parameters

  • shape - tuple representing width and height to crop image to.
  • invert_colors - whether to represent black as 1 and white as 0 in the numpy array.
  • flatten - whether to reshape the numpy array to a single dimension.
  • label - component name in interface.

String Shortcuts

  • "sketchpad" - default implementation.

gradio.inputs.Webcam(shape=(224, 224), label=None)

Component creates a webcam for captured image input. Provides numpy array of shape (width, height, 3) as an argument to the wrapped function.

Parameters

  • shape - tuple representing width and height to crop image to.
  • label - component name in interface.

String Shortcuts

  • "webcam" - default implementation.

gradio.inputs.Microphone(label=None)

Component creates a microphone element for audio imputs. Provides numpy array of shape (samples, 2) as an argument to the wrapped function.

Parameters

  • label - component name in interface.

String Shortcuts

  • "microphone" - default implementation.

Output Components

gradio.outputs.Textbox(self, label=None)

Component creates a textbox to render output text or number.

Parameters

  • label - component name in interface.

String Shortcuts

  • "text" - default implementation.
  • "number" - default implementation.

gradio.outputs.Label(self, num_top_classes=None, label=None)

Component displays a classification labels for an input. Expects a string for the most confident label or a dictionary mapping each string label to a confidence score between 0 and 1 to be returned by the function.

Parameters

  • num_top_classes - number of most confident classes to show.
  • label - component name in interface.

String Shortcuts

  • "label" - default implementation.

gradio.outputs.Image(self, plot=False, label=None)

Component displays an image. Expects a numpy array of shape (width, height, 3)to be returned by the function, or a matplotlib.pyplot if plot = True.

Parameters

  • plot - whether to expect a plot to be returned by the function.
  • label - component name in interface.

String Shortcuts

  • "image" - default implementation.
  • "plot" - sets plot = True.

gradio.outputs.KeyValues(self, label=None)

Component displays a table representing values for multiple fields. Expects a list of 2-length tuples that represent (key, value) pairs to be returned by the function.

Parameters

  • label - component name in interface.

String Shortcuts

  • "keyvalues" - default implementation.
